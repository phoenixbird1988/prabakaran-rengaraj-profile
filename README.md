# R. Prabakaran - Profile Website

A modern, animated profile website showcasing professional experience, skills, and projects.

## Features

- **Modern UI/UX**: Clean, professional design with gradient themes
- **Smooth Animations**: Fade-in, slide-up, and scroll-triggered animations
- **Interactive Elements**: Hover effects, animated skill bars, and interactive timelines
- **Responsive Design**: Fully responsive layout for mobile, tablet, and desktop
- **Smooth Navigation**: Anchor links with smooth scrolling
- **jQuery Animations**: Dynamic animations and interactive features

## Technologies Used

- HTML5
- CSS3 (with animations and transitions)
- jQuery 3.7.1
- Font Awesome 6.4.0 (for icons)

## File Structure

```
Profile/
├── index.html          # Main HTML structure
├── css/
│   └── style.css      # Styling and animations
├── js/
│   └── script.js      # jQuery animations and interactivity
└── README.md          # Project documentation
```

## Sections

1. **Hero Section**: Name, title, and contact information
2. **Career Objective**: Professional goals and aspirations
3. **Profile Summary**: Overview of experience and expertise
4. **Technical Skills**: Animated skill bars for various technologies
5. **Academic Qualification**: Timeline-style education history
6. **Work Experience**: Interactive timeline of professional experience
7. **Work Projects**: Grid layout of projects with hover effects
8. **Achievements**: Notable accomplishments
9. **Personal Details**: Personal information
10. **Contact**: Contact information and footer

## Animations

- Scroll-triggered fade-in animations
- Animated skill progress bars
- Hover effects on cards and buttons
- Smooth scroll navigation
- Parallax effects on hero section
- Back-to-top button with fade animation
- Mobile menu toggle animation

## Responsive Breakpoints

- **Desktop**: Full layout with multi-column grids
- **Tablet** (768px): Adjusted layouts, single column timelines
- **Mobile** (480px): Stacked layouts, optimized spacing

## Browser Support

- Chrome (latest)
- Firefox (latest)
- Safari (latest)
- Edge (latest)

## Usage

Simply open `index.html` in a web browser. No build process or server required.

## Customization

- Colors can be modified in CSS variables (`:root` in `style.css`)
- Animation speeds can be adjusted in CSS transitions
- Content can be updated directly in `index.html`
